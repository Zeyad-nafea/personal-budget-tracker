# Personal Budget Tracker 💰

A simple yet powerful desktop application for tracking your personal finances, built with Python and tkinter. This project evolved from a command-line budget tracker to a modern GUI application with data visualization and user authentication.

## Features ✨

### Core Functionality
* **User Authentication**: Secure login and registration system
* **Budget Management**: Set, update, and monitor your monthly budget
* **Transaction Tracking**: Add and categorize your expenses
* **Category Management**: Create custom spending categories
* **Savings Goals**: Set and track progress toward savings targets

### Analytics & Visualization
* **Interactive Charts**: Visual pie charts showing spending by category
* **Real-time Statistics**: Track total spending and averages
* **Budget Overview**: See remaining budget at a glance
* **Transaction History**: View all transactions with dates and categories

### Data Management
* **Data Export**: Export your financial data to JSON format
* **Persistent Storage**: All data saved locally in text files
* **Data Clearing**: Option to reset all data when needed
* **Multi-user Support**: Individual accounts for multiple users

## Installation 🚀

### Prerequisites
* Python 3.6 or higher
* pip (Python package installer)

### Required Libraries
```bash
pip install matplotlib
```

### Setup
1. Clone this repository:
```bash
git clone https://github.com/yourusername/personal-budget-tracker.git
cd personal-budget-tracker
```

2. Run the application:
```bash
python budget_tracker_gui.py
```

## Usage Guide 📖

### Getting Started
1. On **first launch**, create a new account by entering a username and password, then click **Register**.
2. Next, use your credentials to **Login**.
3. Set your **monthly budget** amount to begin tracking.

### Adding Transactions
1. Click **"Add Category"** to create spending categories (e.g., Food, Transport, Entertainment).
2. Use **"Add Transaction"** to record your expenses.
3. Select the appropriate category and enter the amount.

### Tracking Savings
1. Set savings goals using **"Add Saving Goal"**.
2. Add money to your savings with **"Add to Savings"**.
3. Monitor progress in the **"View Progress"** section.

### Viewing Analytics
* **Overview Tab**: See budget summary and recent transactions.
* **Charts Tab**: Visual representation of spending by category.
* **Recent Tab**: Quick access to your most recent transactions.
